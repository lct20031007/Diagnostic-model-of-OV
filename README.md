# Diagnostic-model-of-OV
 A diagnostic model using SVM for predicting patients' subtypes to aid in clinical diagnosis and treatment.

## Requirements

The main requirements are listed below:

* Python 3.7
* Numpy
* Sklearn
* Pandas
* Joblib
